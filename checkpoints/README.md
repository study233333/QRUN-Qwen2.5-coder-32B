Do not commit full model weights directly to the Git repository.

This directory is intended to provide instructions on how to obtain and place model weights:

1. Local Weight Placement: Place the weights inside the `checkpoints/checkpoints_hybrid_v2/` directory (a placeholder file `epoch_2.pt` is already included).
2. Download Instructions: Please contact the repository maintainers for download links or use private storage services (e.g., S3, GCS). Host only small metadata or download scripts in public repositories.
3. If publishing weights to the Hugging Face Hub, document the upload strategy in `MODEL_CARD.md` and provide the Hugging Face model link in the README.
